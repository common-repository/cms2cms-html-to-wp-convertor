Automated HTML to WordPress content migration plugin. Convert HTML to WordPress easily and fast.

== Description ==

You have a lot of content on your HTML website but you considered that WordPress is a better place to fit in. You start thinking of moving HTML website content to WordPress. You’re mostly interested in a quick and profound migration.

= HOW THIS PLUGIN HELPS: =
- Free Demo Migration helps see the migration outcome beforehand;
- 15 min is the average migration time; Note: timing depends on how many content items you’re going to move from HTML to WordPress;
- Migrate the content you want. Choose how you want it to be and look on WordPress; (Data ‘Mapping’ option);
- Complete data migration price depends on the content volume you’re about to migrate to WordPress.

= WHAT WILL BE MOVED FROM HTML TO WORDPRESS? =
*(Note that CMS2CMS moves only content data from HTML to WordPress)*
* Pages along with the content included there;
* Attachments (images and videos);
* Images included in the content body.


= PRESERVED SEO JUICE AFTER THE MIGRATION =
* Migration of keywords, metadata/meta description;
* Automated 301 redirects from existing HTML website to the new WordPress website.


= COMPATIBLE WITH: WORDPRESS 4.1.X - 5.0.X. =
- Live help is available. Our support team is here 24/7.
- When a custom migration is needed, CMS2CMS offers data migration service packages to order.


== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose the downloaded file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS: Automated HTML To WordPress Migration Plugin”.
5. Register an account or sign in to the existing one.
6. Provide your Existing website URL address.
7. Click ‘Verify Connection’. You’ll be redirected to the CMS2CMS website to proceed with your migration.
